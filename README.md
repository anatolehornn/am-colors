# am colorschemes


### color schemes
 - amlight.vim
 - amdark.vim    
 - amcolors.vim:
   combines amlight and amdark
 - ampresent.vim:
   light vim colorscheme optimized for presentations

### plugin support:
 - CleverF         
 - Flashy         
 - indent line 
 - indent guides
 - braceless 
 - signify
 - ALE (Asynchronous Lint Engine)
 - GitGutter
 - multi-cursor
 - tagbar
 - find & replace
 - CtrlP
 - startify 
 - poppy
 - easytags

### special file type support
 - xml
 - html
 - json
 - Mathematica (mma)
